📊 Como receber respostas no Google Sheets

  1. 1
    No arquivo HTML, localize a linha const DEST = 'SEU-EMAIL-AQUI' e substitua pelo seu e-mail real.
  2. 2
    Hospede o arquivo em qualquer servidor (ex: GitHub Pages — gratuito) ou use o Live Server do VS Code para testes.
  3. 3
    Na primeira resposta, o FormSubmit envia um e-mail de ativação. Clique em "Confirm your submission".
  4. 4
    Acesse formsubmit.co → faça login → vá em Integrations → Google Sheets e cole o link da sua planilha.
  5. 5
    Pronto! Cada envio gera uma nova linha na planilha com: data, nome, e-mail, pontuação e todas as respostas. ✅
